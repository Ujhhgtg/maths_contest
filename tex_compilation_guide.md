# Tex Compilation Guide

## Windows

### Prepare

1. Install [scoop](https://scoop.sh)
2. `scoop install miktex`

### Compile

1. `xelatex ./论文正文.tex` and hit install for all package installation prompted

## Arch Linux

### Prepare

1. `yay -S texlive-meta texlive-langchinese`

### Compile

1. `xelatex ./论文正文.tex`
